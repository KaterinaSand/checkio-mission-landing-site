The size of the hexagonal grid is 12x9 ('A1' to 'L9') with this mission.

  • It’s characteristics:
    • the alphabet letters represent columns and numbers represent rows;
    • it’s flat topped;
    • the even columns are being shoved down.

You are given a set of obstacle hexes on the moon as input value. You have to return a set of all candidates for the landing site.

  • The conditions of landing site hex:
    • The hex isn't obstacle hex.
    • No obstacle hexes in equidistant ( 1 or more ) hexes.
    • The above radius is the longest.
NOTE:
  • Outside the hexgonal grid is always considered an obstacle hex.
  • If there is no landing site, return the empty set.

Example: 

assert landing_site({'E5', 'E7', 'F4', 'F6', 'G4', 'G6', 'H3', 'H5'}) == {'C3', 'J7'}
assert landing_site({'A4', 'C2', 'C6', 'C9', 'D4', 'D7', 'F1', 'F5',
                     'F8', 'G4', 'H7', 'I2', 'I5', 'I9', 'K3', 'K8', 'L5'}) == {'B7', 'E3', 'J6'}

Input: obstacles on the moon (set of string)

Output: a set of all candidates for the landing site (set of string)

Precondition:

  • all(re.fullmatch('[A-L][1-9]', i) for i in input)

How it is used: You can find the area you need.
